SimpleSpring
============

This is a Simple Spring MVC application

##Getting Started##

###Requirements###

Maven

###Steps###

Clone the project repository using git

##Deployment##

* mvn tomcat7:run 

This packages the war and deploys the application on to
the embedded tomcat server (mentioned in pom) 

application url: **http://localhost:9966/simplespring/**

alternatively, you can build the war and deploy onto external tomcat server.
